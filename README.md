# Personal Portfolio

This is my **personal portfolio website**, designed to showcase my skills, projects, and experience as a web developer.  
It features a **modern dark theme**, smooth animations, and a fully **responsive design** for desktop and mobile devices.

## Tech Stack
- **React (Vite)** – For fast and reactive frontend
- **Tailwind CSS** – For modern, minimalistic styling
- **Framer Motion** – For smooth animations and interactive effects

## Features
- Hero section with introduction
- About me section with skills and experience
- Projects showcase with interactive hover effects
- Contact form for inquiries
- Fully responsive and fast loading
